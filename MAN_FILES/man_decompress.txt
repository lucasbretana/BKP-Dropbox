NAME
  decompress - shell function that automaticlly decompress some know files

DESCRIPTION
  decompress is a shell script function that automates the process of decompress files based on more common know filne types.

  External tools
    This function uses other tools that must be installed, such as unrar, unzip, tar, bunzip2.

OPTIONS
  -bz2 forces to bunzip only, using bunzip2

    $ decompress -bz2 filepath.bz2
  
  -tar.bz2 forces to bunzip and untar, using tar

    $ decompress -bz2 filepath.tar.bz2
  
  -tar.gz forces to untar with -z

    $ decompress -tar.gz filepath.tar.gz
  
  -tgz forces to untar with -z

    $ decompress -tgz filepath.tgz
  
  -gz forces to gunzip only

    $ decompress -gz filepath.gz
  
  -tar forces to untar only

    $ decompress -tar filepath.tar
  
  -tar.xz forces to gunzip then untar, using tar

    $ decompress -tar.xz filepath.tar.xz
  
  -zip forces to use the unzip to decompress

    $ decompress -zip filepath.zip

  -7z forces to unzip using 7z

    $ decompress -7z filepath.7z

  -rar forces to decompress uszing unrar

    $ decompress -rar filepath.rar

SEE ALSO
  tar(1), unzip(1), unrar(1), bunzip2(1), 7z(1).

BUGS
  Only decompress some common files types.
  This manual is more complex than the function.
