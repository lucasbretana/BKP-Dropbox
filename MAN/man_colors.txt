NAME
	colors - prints all know colors

SYNOPSIS
	colors [string]

DESCRIPTION
	colors prints a given string in all know colors, or a default string "Hello World"

OPTIONS
	colors takes no optins

EXAMPLE

	$ colors
	$ colors "Welcome to hell"
